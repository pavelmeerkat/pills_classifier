# Классификатор таблеток

### Проект представляет собой нейросетевой классификатор для распознавания типов таблеток по изображениям.


## Метрики качества
- Целевая точность: ≥ 75%
- Используемые метрики: Accuracy, Precision, Recall, F1-score

## Скачивание датасета
wget https://disk.yandex.ru/d/o4oQ4Yk7SpH1dw -O ogyeiv2.zip

## Распаковка
unzip ogyeiv2.zip -d dataset
